# Personal Site

A simple, minimal site: a home/about page, a resume page, a projects
page, and a read-only diary page, sharing one stylesheet. Built to run
as-is on GitHub Pages.

## Files

- `index.html` — Home + About
- `resume.html` — Resume
- `projects.html` — Projects
- `garden-leave.html` — Garden Leave 2026 (a read-only diary)
- `style.css` — shared styles for all four pages

## Publishing to GitHub Pages

1. Create a new GitHub repository (or use an existing one).
2. Add these five files to the repository root.
3. In the repo, go to **Settings → Pages**.
4. Under **Build and deployment → Source**, choose **Deploy from a
   branch**, then pick your default branch (usually `main`) and the
   `/ (root)` folder. Save.
5. GitHub will give you a URL, usually
   `https://yourusername.github.io/your-repo-name/`, live within a
   minute or two.

If you want the site at the root of `yourusername.github.io` (no repo
name in the URL), name the repository exactly `yourusername.github.io`.
